Jawaban Soal Nomor 1

Cara Kerja Web
Sebuah website berfungsi sebagai kumpulan halaman web yang dapat diakses melalui internet. Proses kerja dimulai ketika pengguna memasukkan alamat URL atau mengklik tautan untuk mengakses situs web. Permintaan tersebut dikirimkan ke web server yang meng-host situs web tersebut. Web server, sebagai perangkat keras atau perangkat lunak, menanggapi permintaan tersebut dengan mengirimkan balasan berupa data halaman web yang diminta. Data ini kemudian di-render oleh browser pengguna untuk ditampilkan sebagai halaman web yang interaktif. Proses ini terjadi berkat kerjasama antara domain, yang memberikan alamat unik untuk mengidentifikasi lokasi situs web, hosting yang menyimpan semua file dan data situs, dan web server yang menanggapi permintaan pengguna dan mengirimkan konten yang sesuai.

Hubungan antara Web Server, Domain dan Hosting
Domain, hosting, dan web server merupakan komponen kunci dalam ekosistem situs web. Domain memberikan alamat yang mudah diingat untuk mengarahkan pengguna ke server tempat situs web di-host. Hosting menyediakan ruang penyimpanan untuk file dan data situs. Web server, sebagai pemroses utama, menanggapi permintaan pengguna dengan mengirimkan konten situs. Kerjasama ketiganya memungkinkan situs web dapat diakses secara efisien melalui internet, memberikan pengguna pengalaman interaktif yang dihasilkan dari interaksi antara browser dan server.
